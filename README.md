## 📂 Key Features
- **Dynamic Popup:** 별도의 HTML 파일 없이 JS 객체(`projectData`)를 활용한 데이터 동적 렌더링 팝업 구현.
- **Auto Deployment:** GitHub Actions를 연동하여 `main` 브랜치 푸시 시 자동 배포 자동화.
- **Brutalism UI:** 굵은 선과 강렬한 대비를 활용한 독창적인 디자인 시스템 구축.
